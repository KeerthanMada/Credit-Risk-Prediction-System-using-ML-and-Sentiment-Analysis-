# 📊 Credit Risk Prediction System using ML & Sentiment Analysis

## 🚀 Project Overview
This project presents a **Smart Credit Risk Prediction System** that uses **Machine Learning + Sentiment Analysis** to classify loan applicants as:

- ✅ Good Credit Risk  
- ❌ Bad Credit Risk  

It combines **financial data + loan officer remarks** to improve prediction accuracy and decision-making.

---

## 🧠 Key Features
- 🔍 **Machine Learning Model** – Extra Trees Classifier  
- 💬 **Sentiment Analysis** – TextBlob (Polarity & Subjectivity)  
- 📊 **Explainable AI** – SHAP for feature-level explanations  
- 🌐 **Web App** – Built using Streamlit  
- 📈 **EDA Visualizations** – Insights from dataset

## 📊 Model Performance
- Accuracy: ~78–82%  
- ROC-AUC: > 0.80  
- Strong classification performance on German Credit Dataset  

---

## 💡 How It Works
1. Input applicant details  
2. Enter loan officer remarks  
3. Sentiment analysis extracts:
   - Polarity (-1 to +1)  
   - Subjectivity (0 to 1)  
4. Model predicts credit risk  
5. SHAP explains prediction
